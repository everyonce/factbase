# Modal Jazz

Modal jazz: A style of jazz improvisation based on musical modes (scales) rather than rapidly changing chord progressions — emerged prominently with Miles Davis's "Kind of Blue" (1959).

## Key Characteristics

- Improvisation over static or slowly changing modes @t[1959..] [^1]
- Fewer chord changes per bar than bebop @t[?]
- Allows more melodic freedom for the soloist @t[?]
- Associated with Miles Davis, John Coltrane, Bill Evans @t[1959..1965] [^1]

## Common Modes Used

- Dorian mode (minor with raised 6th) @t[?]
- Mixolydian mode (major with lowered 7th) @t[?]
- Phrygian mode (minor with lowered 2nd) @t[?]

## Historical Context

- Reaction against bebop's dense harmonic movement @t[~1958] [^2]
- George Russell's "Lydian Chromatic Concept" provided theoretical basis @t[=1953] [^3]
- John Coltrane later extended modal approach into "sheets of sound" @t[~1960] [^4]
