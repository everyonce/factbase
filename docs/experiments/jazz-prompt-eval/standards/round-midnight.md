# Round Midnight

## Overview

- Composed by Thelonious Monk @t[=1944] [^1]
- Originally titled "'Round About Midnight" @t[=1944] [^1]
- Monk's most recorded composition @t[?]
- Cootie Williams recorded the first version @t[=1944] [^2]

## Harmonic Character

- Dark, chromatic harmonies characteristic of Monk's style @t[?]
- Uses tritone substitutions extensively @t[?]
- AABA form, 32 bars @t[?]

## Notable Recordings

- Miles Davis, "Round About Midnight" (Columbia CL 949) @t[=1957] [^3]
- Thelonious Monk Quartet, "Brilliant Corners" @t[=1957] [^4]
- Herbie Hancock, "Speak Like a Child" @t[=1968] [^5]

## Disputed Attribution

- Some sources credit Cootie Williams as co-composer @t[?] [^6]
- Monk maintained sole authorship throughout his life @t[?]
